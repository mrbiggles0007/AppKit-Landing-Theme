Cloud Integration

Both Unitrends Recovery 926S and Infrascale ES 4550 offer cloud integration options. Similarly, Infrascale ES 4550 offers cloud integration with multiple cloud platforms such as Amazon S3, Microsoft Azure, Google Cloud, and others. The appliance's cloud integration also enables businesses to implement a hybrid cloud backup strategy that combines on-premise backup with cloud backup.
